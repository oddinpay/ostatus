# worker-template-go

## Requirements

- Node.js
- Go 1.24.0 or later

## Getting Started

### Commands

```
pnpm run dev      # run dev server
# or
go run .       # run dev server without Wrangler (Cloudflare-related features are not available)
pnpm run build  # build Go Wasm binary
pnpm run deploy # deploy worker
```
